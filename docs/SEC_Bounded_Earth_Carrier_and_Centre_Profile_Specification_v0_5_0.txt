STRUCTURAL EARTH CENTRE
BOUNDED EARTH CARRIER AND CENTRE PROFILE SPECIFICATION
Release v0.5.0

1. PURPOSE

The bounded Earth layer maps recognizable Earth-centre questions into explicit claim dependencies and resolution states.

It is a claim-boundary layer, not a universal geodetic model.

2. EARTH PROFILE PRINCIPLE

An Earth-centre claim is not admitted merely because a coordinate can be displayed.

The profile must declare the carrier and the dependencies material to the claim.

8. AUTHORITY

The area-vector result is not silently promoted to:

physical geocentre
reference-frame origin
population centre
gravity centre
universal geographic centre

The boundary-vector result receives the same authority limitation.

Each is a bounded result under its own profile.

9. NUMERICAL POLICY

Both real-land vector profiles declare 9-decimal identity quantization for identity-bearing floating-point result fields.

This policy is separate from the exact rational core.

10. DATA BOUNDARY

The included dataset is a frozen input artifact.

A different dataset hash defines a different data boundary and does not automatically inherit the current result identities.

11. RELATION TO STRUCTURAL INNOVATION LAYER

The Earth layer supplies domain-specific claim examples.

The structural innovation layer supplies reusable mechanisms for:

claim comparison
stability regions
symmetry witnesses
resolution certificates

The two layers remain separable so that Earth-specific assumptions do not become universal structural rules.
